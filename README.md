# SGA – Sistema de Gerenciamento da Automação

**Progressive Web App (PWA)** para gestão das solicitações de Automação da ArcelorMittal Brasil – Mina do Andrade.

## Módulos
- **SGF** – Sistema de Gerenciamento de Forces
- **SAL** – Solicitação de Alteração de Lógicas  
- **SGP** – Sistema de Gerenciamento do PIMS
- **SCA** – Sistema de Controle de Acesso

## Acesso controlado
O acesso é gerenciado pelo Supervisor de Automação. Usuários solicitam acesso via formulário e aguardam aprovação.

## Instalar no smartphone
1. Abra o link no Chrome (Android) ou Safari (iOS)
2. Toque em **"Adicionar à tela inicial"** / banner de instalação
3. O app funciona offline após a primeira abertura
