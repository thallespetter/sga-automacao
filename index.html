Conectar ao Firebase
O SGA usa Firebase para sincronização em tempo real entre todos os dispositivos. Configure uma vez e todos os dados ficam na nuvem.
1. Criar projeto Firebase (gratuito)Acesse console.firebase.google.com → "Adicionar projeto" → desative Google Analytics → Criar
2. Ativar Realtime DatabaseNo menu lateral: Build → Realtime Database → Criar banco → Escolha região → Modo de teste (30 dias)
3. Registrar o app WebConfigurações do projeto (⚙️) → Adicionar app → Web (</>) → apelido "SGA" → Registrar → copie o objeto firebaseConfig
Salvo localmente. Você só configura uma vez por dispositivo.
